# 🇧🇫 Bara Concours

> Application de préparation aux concours directs du Burkina Faso

**Bara Concours** est une PWA (Progressive Web App) développée par **Bara Formation** pour accompagner les candidats burkinabè dans leur préparation aux concours directs de la fonction publique.

## 🎯 Fonctionnalités

- 📅 **108 concours officiels session 2026** (Communiqué N°26-00351/MSP/SG/AGRE/DOC)
- 📚 **2 600+ questions QCM** dans 72 matières
- 🎓 **3 niveaux** : BEPC, BAC, Licence
- 📝 **Mode entraînement** : Sujets de 25 questions avec correction immédiate
- ⏱️ **Mode examen chronométré** : 50 questions en 1 heure, avec sauvegarde
- 📊 **Statistiques** par matière, suivi de progression
- 💬 **Forum** communautaire entre candidats
- 📖 **Accompagnement Final** : annales corrigées (Premium)
- 🔐 **Authentification** SMS + Empreinte digitale
- 🌐 **Mode hors-ligne** : fonctionne sans internet

## 🌟 Niveaux et concours couverts

| Niveau | Nombre de concours |
|---|---|
| BAC | 54 concours |
| Licence | 31 concours |
| BEPC | 20 concours |
| CEPE/CAP | 3 concours |

Plus 3 concours par autres canaux (Police, Gendarmerie, Sapeurs-Pompiers).
