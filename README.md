# 🇧🇫 Bara Concours

> Application officielle de préparation aux concours de la fonction publique du Burkina Faso

**Bara Concours** est une PWA (Progressive Web App) développée par **Bara Formation** pour accompagner les candidats burkinabè dans leur préparation aux concours de la fonction publique.

🌐 **App en ligne** : [bara-formation.github.io/bara-concours](https://bara-formation.github.io/bara-concours/)

---

## 📊 En chiffres

| Métrique | Valeur |
|---|---|
| 🎯 **Questions QCM** | **5 115+** dans la banque |
| 📚 **Matières couvertes** | **37 matières** spécialisées |
| 🏛️ **Concours session 2026** | **108 concours directs** |
| 🎓 **Niveaux** | CEPE/CAP, BEPC, BAC, Licence |
| 📅 **Dates d'épreuves 2026** | 15 au 22 juillet 2026 |
| ⚡ **Distribution corrigés** | Instantanée (<30 sec via cloud) |

---

## 🎯 Fonctionnalités principales

### Pour les candidats
- 📅 **108 concours officiels session 2026** avec dates et lieux d'épreuves
- 📚 **5 115+ questions QCM** avec corrections détaillées
- 📝 **Mode entraînement** : QCM par matière avec correction immédiate
- ⏱️ **Examen hebdomadaire** : 50 questions chronométrées le dimanche 19h30-20h30
- 🏆 **Classement** national des candidats (Premium)
- 📊 **Statistiques** détaillées par matière et suivi de progression
- 💬 **Forum communautaire** entre candidats (Firestore temps réel)
- 🎓 **Accompagnement Final** : sessions Premium avec corrigés détaillés
- 🔐 **Authentification** Google / Email
- 🌐 **Mode hors-ligne** : fonctionne sans internet une fois chargé
- 📱 **PWA installable** : ajout à l'écran d'accueil du téléphone

### Pour l'administration
- ⚙️ Tableau de bord admin complet
- 📥 Import de QCM par lots avec parser automatique
- 🎨 CMS textes éditables Firestore (distribution instantanée)
- 🎓 Publication de sessions Accompagnement Final en direct
- 💳 Génération et suivi de codes Premium
- 📈 Analytics utilisateurs et Premium

## 🛠️ Technologies

- **Frontend** : HTML5, CSS3, JavaScript vanilla (no framework)
- **Backend cloud** : Firebase (Auth + Firestore)
- **PWA** : Service Worker, Manifest, mode hors-ligne
- **Distribution** : GitHub Pages
- **Design** : Charte 🇧🇫 (vert, jaune, rouge) + typographie Instrument Serif
