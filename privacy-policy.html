Politique de Confidentialité

Dernière mise à jour : 18 mai 2026

Bara Formation respecte votre vie privée. Cette politique explique quelles informations nous collectons via l'application Bara Concours, comment nous les utilisons, et vos droits.

1. Informations collectées

L'application Bara Concours collecte les informations suivantes :

  • Numéro de téléphone : utilisé pour créer votre compte et envoyer les codes de vérification SMS.
  • Nom (optionnel) : que vous fournissez lors de l'inscription pour personnaliser votre expérience.
  • Ville (optionnel) : pour localiser géographiquement les utilisateurs.
  • Concours visé : pour personnaliser le contenu affiché.
  • Historique de QCM : vos scores et progression, stockés uniquement sur votre appareil.
  • Empreinte digitale (optionnel) : utilisée pour la connexion biométrique. Cette donnée ne quitte JAMAIS votre téléphone.

2. Utilisation des informations

Vos informations sont utilisées exclusivement pour :

  • Vous authentifier de manière sécurisée
  • Sauvegarder votre progression et vos résultats
  • Vous envoyer des notifications de révision (uniquement si vous l'acceptez)
  • Améliorer l'expérience utilisateur

Nous ne vendons JAMAIS vos données. Nous ne partageons rien avec des tiers à des fins commerciales.

3. Stockage des données

La grande majorité de vos données sont stockées localement sur votre téléphone (localStorage). Si vous activez la synchronisation cloud (optionnelle), les données suivantes sont stockées sur les serveurs Firebase de Google :

  • Numéro de téléphone
  • Profil (nom, ville, concours visé)
  • Historique des scores

Ces serveurs respectent les standards internationaux de sécurité.

4. Authentification biométrique

Si vous activez la connexion par empreinte digitale, sachez que :

  • Votre empreinte est traitée uniquement par le système d'exploitation de votre téléphone
  • L'application Bara Concours ne reçoit JAMAIS votre empreinte digitale
  • L'application reçoit uniquement une confirmation cryptographique que vous êtes l'utilisateur autorisé
  • Vous pouvez désactiver cette fonction à tout moment dans les paramètres

5. Vos droits

Vous avez le droit de :

  • Accéder à vos données personnelles
  • Demander la suppression de votre compte (en nous contactant)
  • Désactiver les notifications à tout moment
  • Désactiver la biométrie à tout moment
  • Refuser la synchronisation cloud

6. Sécurité

Nous prenons des mesures techniques pour protéger vos données :

  • Transmission chiffrée HTTPS
  • Pas de stockage de mots de passe (utilisation de SMS et biométrie)
  • Sécurité Google Firebase pour la synchronisation cloud

7. Mineurs

L'application est destinée aux candidats des concours directs au Burkina Faso, généralement âgés de 18 ans et plus. Les mineurs souhaitant utiliser l'application doivent obtenir l'accord de leurs parents ou tuteurs.

8. Modifications de cette politique

Cette politique peut être mise à jour. La date de "Dernière mise à jour" en haut de cette page indique la version actuelle. Les changements majeurs vous seront communiqués via l'application.
